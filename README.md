This project is a scalable web application designed to streamline the recruitment process by providing dedicated dashboards for Admin, Recruiters, and Students.

It implements secure Role-Based Access Control (RBAC), ensuring that users can only access features relevant to their roles after authentication.

The Admin manages users and monitors system activity, Recruiters post job opportunities and review applications, while Students can apply for jobs and track their application status.

The system uses RESTful APIs to enable smooth communication between the frontend and backend, ensuring efficient data handling and performance.

 Tech Stack
-> Frontend: React.js, Tailwind CSS
-> Backend: Spring Boot (Java)
